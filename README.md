#AI Travel Planner

An intelligent AI-powered travel planning web application that helps users explore destinations, generate structured travel itineraries, and estimate budgets for individuals or groups.

🚀 Features
  📍 Destination Insights
  Get detailed information about any travel destination including best time to visit, culture, and highlights.
  
  ⭐ Top Attractions
  Discover must-visit places with curated recommendations.
  
  🏨 Accommodation Suggestions
  Budget, mid-range, and luxury hotel options with estimated pricing.
  
  🍽️ Food & Local Experience
  Explore local cuisines and unique cultural experiences.
  
  🚗 Transportation Guide
  Get details on flights, trains, and local transport.
  
  💰 Budget Planner
  Calculate estimated travel cost per person and for groups.
  
  👥 Group Travel Support
  Automatically adjusts budget based on number of travelers.
  💬 Interactive Chat Interface
  Ask queries like:
  “Plan a trip to Goa”
  “Budget for 4 people in Paris”
  
🛠️ Tech Stack
  Frontend & Backend: Streamlit
  AI Integration: Groq API
  Language: Python
  Libraries: streamlit, groq, python-dotenv
  
🔐 Security
  API keys are stored securely using environment variables (.env)
  Sensitive files are excluded using .gitignore
  No API keys are exposed in the repository
  
📦 Installation

  1️⃣ Clone the repository
  git clone https://github.com/your-username/travel-chatbot.git
  cd travel-chatbot
  
  2️⃣ Install dependencies
  python -m pip install -r requirements.txt
  
  3️⃣ Setup environment variables
  
  Create a .env file:
  
  GROQ_API_KEY=your_api_key_here
  
  4️⃣ Run the application
  python -m streamlit run travel_chatbot.py
  📸 Sample Usage
  “Tell me about Bali”
  “Plan a 5-day trip to Paris”
  “Budget for 3 people in Goa”
